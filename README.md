# 🧹 DecodeLabs Internship - Week 1: Data Cleaning & Preparation

## 📌 Project Overview

This repository contains my Week 1 internship task completed during the **DecodeLabs Data Analytics Internship (Batch 2026)**.

The objective of this project was to clean and prepare a raw e-commerce dataset by identifying data quality issues and transforming the dataset into a clean, analysis-ready format.

---

## 🎯 Objectives

- Identify and handle missing values
- Detect and remove duplicate records
- Standardize data formats
- Validate data consistency
- Prepare the dataset for Exploratory Data Analysis (EDA)

---

## 🛠️ Tools & Technologies

- Microsoft Excel
- Data Cleaning Techniques
- Data Validation
- Git & GitHub

## 📂 Dataset Information

The dataset contains e-commerce transaction records with features such as:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source
- Total Price

---

## ✅ Data Cleaning Tasks Performed

### 1. Missing Values
- Identified missing values in the **CouponCode** column.
- Replaced blank coupon values with **"NONE"** because not every customer uses a coupon.

### 2. Duplicate Records
- Checked the entire dataset for duplicate rows.
- Verified duplicate Order IDs.
- No duplicate records were found.

### 3. Data Formatting
- Standardized date formatting.
- Formatted Unit Price and Total Price columns.
- Ensured consistent text formatting across categorical columns.

### 4. Data Validation
- Verified Quantity values.
- Verified Unit Price values.
- Verified Total Price values.
- Confirmed data consistency across all records.

### 5. Documentation
Created a documentation sheet summarizing:
- Missing values handled
- Duplicate verification
- Formatting changes
- Cleaning summary

## 📊 Results

- Dataset successfully cleaned.
- Missing values handled.
- Duplicate records removed/verified.
- Consistent formatting applied.
- Dataset prepared for Exploratory Data Analysis (EDA).
